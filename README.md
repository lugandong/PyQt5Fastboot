### 简介
这是一款Android刷机软件，封装了fastboot操作，界面采用Qt5.5.1，基于Python3.4版本进行开发

### 使用方法
`python main.py`

### Py2exe打包
基于Python3.4 Py2exe 64bit软件下载地址[Py2exe 64bit Python3.4](http://download.csdn.net/detail/djstavav/9322751)

`python setup.py`
